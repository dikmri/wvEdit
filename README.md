# wvEdit

ブラウザで動作する軽量ノンリニア動画編集Webアプリ。インストール不要・無料・ローカル処理完結。

🔗 **[GitHub Pagesで開く](https://dikmri.github.io/wvEdit/)**

## 特徴

- 動画ファイルをサーバーへアップロードしない（ローカル処理）
- MP4動画のカット・削除・移動・トリミング
- BGM/SE音声トラック対応
- 自由配置テキスト（X/Y位置・回転）
- 字幕オーバーレイ（下部固定・背景付き）
- 画像クリップ対応
- ffmpeg.wasmによるMP4書き出し（YouTube/Shorts/SNSプリセット）
- プロジェクトのJSON保存/復元
- Undo/Redo対応

## 対応環境

- Google Chrome 最新版（推奨）
- Microsoft Edge 最新版

## 使い方

### 素材の読み込み

1. 素材パネルの「+ 追加」ボタン、またはドラッグ&ドロップで動画・音声・画像を読み込む
2. タイムラインへダブルクリックまたはドラッグ&ドロップで配置

### テキスト・字幕

- 素材パネルの **「T+」** ボタン → 自由配置テキストを追加
  - インスペクターでX/Y位置（%）・回転角度・フォント・色を調整
- 素材パネルの **「字+」** ボタン → 字幕を追加
  - インスペクターで縦位置（上部/中央/下部）・フォント・色を調整

### 再生・編集

| 操作 | 内容 |
| --- | --- |
| Space | 再生/停止 |
| S | 再生ヘッド位置で分割 |
| Delete / Backspace | 選択クリップを削除 |
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
| Ctrl+S | プロジェクト保存（JSON） |
| ← / → | 前/次フレーム |
| Shift+← / Shift+→ | 1秒戻る/進む |

### 書き出し

「書き出し」ボタンから以下のプリセットで MP4 書き出しが可能：

| プリセット | 解像度 | FPS |
| --- | --- | --- |
| プロジェクト設定 | プロジェクト依存 | プロジェクト依存 |
| YouTube | 1920×1080 | 30fps |
| Shorts / 縦動画 | 1080×1920 | 30fps |
| SNS 正方形 | 1080×1080 | 30fps |

## トラック構成

| トラック | 用途 |
| --- | --- |
| テキスト | 自由配置テキスト（X/Y%・回転対応） |
| 字幕 | 横中央固定・下部表示の字幕（背景あり） |
| 映像 | 動画・画像クリップ |
| 音声 | BGM・SE音声クリップ |
